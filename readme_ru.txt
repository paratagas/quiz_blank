Программа с тестовыми заданиями
Версия от 03.12.2016
Программа не требует установки сервера
Для запуска программы нужно запустить файл index.html в браузере

Выскакивающие информационные окошки можно двигать, удерживая за заголовок

Для редактирования вопросов откройте questions.js и следуйте инструкциям
Для редактирования элементов дизайна страницы откройте design.js и следуйте инструкциям
